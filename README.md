# Installation

## Install zsh

`brew install zsh`

## Install zplug

`brew install zplug`

## Install fzf

```
brew install fzf

# To install useful key bindings and fuzzy completion:
$(brew --prefix)/opt/fzf/install
```

## Load it
Move zshrc to `~/.zshrc`

`source .zshrc`
